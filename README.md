## 特性
- 简洁 (~25kB gzipped)
- 有趣
- 自托管
- Markdown
- 自定表情 #(滑稽)
- 管理员认证，防冒名
- 评论打分

## 食用
[查看 DEMO](https://artalk.js.org) / [查看代码](./demo/index.html)

## 开发

### Build

```bash
git clone https://github.com/qwqcode/Artalk.git
cd Artalk
yarn install
yarn run build
```

### Dev

```bash
git clone https://github.com/qwqcode/Artalk.git
cd Artalk
yarn install
yarn run dev
```
