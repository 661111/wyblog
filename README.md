# Artalk
> 一款有趣有后端的评论系统

## 特性
- 简洁
- 有趣
- 自管理
- Markdown
- 自定表情 #(滑稽)
- 评论打分

## 食用
[查看DEMO](./demo/index.html)

## 开发

### Build

```bash
git clone https://github.com/qwqcode/Artalk.git
cd Artalk
yarn install
yarn run build
```

### Dev

```bash
git clone https://github.com/qwqcode/Artalk.git
cd Artalk
yarn install
yarn run dev
```
